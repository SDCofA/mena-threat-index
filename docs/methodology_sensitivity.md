## Governance boundary

This table is a deterministic one-at-a-time sensitivity diagnostic, not a
forecast benchmark or evidence of comparative accuracy. It does not alter any
recorded score. Its fixed fixture, tested parameters, and generated values must
be kept together when the results are cited.

`python -m scripts.forecast_backtest --json` emits the current rolling-origin
diagnostic together with its sample count, period, horizon, exclusions,
missing-data treatment, per-model results, limitations, code commit, data
snapshot hash, and benchmark-manifest hash/status. The legacy text command and
score calculations are unchanged. The report deliberately keeps
`comparativeClaimEligible` false: no public accuracy comparison is permitted
until a preregistered frozen benchmark includes the required baselines and its
paired bootstrap 95% confidence interval excludes no improvement.
